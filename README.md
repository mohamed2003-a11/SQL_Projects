Car Insurance Database
A database project that simulates a car insurance system. It includes scripts to create tables (customers, vehicles, policies, claims) and insert sample data.
I also used XML and XSL for structured data formatting.

Data Cleaning Project
A script that cleans a messy layoffs dataset by removing duplicates, fixing formatting issues, handling missing values, and improving data types.

Exploratory Data Analysis
Queries used to analyze the cleaned data, focusing on trends like layoffs by industry, company size, and time. 
It also uses advanced SQL features like CTEs and window functions.
